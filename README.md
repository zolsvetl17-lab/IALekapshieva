**Автор:** Алина Лекапшиева

## Описание программы

Book Tracker — это графическое приложение для учёта прочитанных книг. Позволяет добавлять книги, фильтровать их по жанру и количеству страниц, сохранять данные в формате JSON.

### Функциональность:
* Добавление книг с указанием названия, автора, жанра и количества страниц.
* Отображение списка книг в виде таблицы.
* Фильтрация по жанру и количеству страниц.

## Инструкция по запуску
### Требования:

* Python 3.6 или выше;

* стандартные библиотеки Python: tkinter, json, os (устанавливать не нужно).

### Шаги для запуска:

1. Клонируйте репозиторий:
```
git clone https://github.com/zolsvetl17-lab/IALekapshieva.git
cd Book-Tracker
```
2. Убедитесь, что у вас установлен Python 3.6+. Проверить версию можно командой:
```
python --version
или
python3 --version
```
3. Запустите приложение:
```
python book_tracker.py
(или python3 book_tracker.py на macOS/Linux)
```
## Примеры использования
### **Сценарий 1: Добавление книги**
1.	Заполните поля:
*	**Название книги**: «1984»;
*	**Автор**: «Джордж Оруэлл»;
*	**Жанр**: «Антиутопия»;
*	**Количество страниц**: 328.
2.	Нажмите кнопку **«Добавить книгу»**.
3.	Книга появится в таблице внизу окна.
 	
### **Сценарий 2: Фильтрация по жанру**
1.	В выпадающем списке **«Фильтр по жанру»** выберите «Антиутопия».
2.	Нажмите **«Применить фильтры»**.
3.	В таблице останутся только книги жанра «Антиутопия».

### **Сценарий 3: Фильтрация по количеству страниц**
1.	В поле **«Фильтр страниц (>)** введите 200.
2.	Нажмите **«Применить фильтры»**.
3.	В таблице останутся книги с количеством страниц больше 200.

## **Сценарий 4: Комбинированная фильтрация**
1.	Выберите жанр «Фантастика» в фильтре по жанру.
2.	Введите 300 в поле «Фильтр страниц (>)`.
3.	Нажмите **«Применить фильтры»**.
4.	В таблице появятся только книги жанра «Фантастика» с количеством страниц > 300.
