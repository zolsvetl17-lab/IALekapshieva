# Currency Converter

**Автор:** Лекапшиева Алина


## Описание

Приложение для конвертации валют с использованием актуальных курсов через внешний API. Включает функционал сохранения истории конвертаций и удобный графический интерфейс.

## Функциональность


* Конвертация между различными валютами.
* Получение актуальных курсов валют через API.
* Сохранение истории последних 10 конвертаций.
* Валидация ввода (проверка на положительное число).
* Удобный графический интерфейс на базе tkinter.
